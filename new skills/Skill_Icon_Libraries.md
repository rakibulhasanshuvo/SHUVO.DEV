# Skill: Icon Libraries

This skill covers the use of inline SVGs packaged as functional React components to ensure strict tree-shaking and small bundle sizes.

## Resources

### 1. Heroicons
*   **URL**: [https://heroicons.com/](https://heroicons.com/)
*   **Licensing**: MIT License.
*   **Usage**: De facto standard for developers operating extensively within the Tailwind ecosystem.

### 2. Lucide React
*   **URL**: [https://lucide.dev/](https://lucide.dev/)
*   **Licensing**: ISC License.
*   **Usage**: Fork and community-driven evolution of Feather Icons. Exposes highly granular props.

### 3. Hugeicons
*   **URL**: [https://hugeicons.com/](https://hugeicons.com/)
*   **Licensing**: Free Tier (Unlimited personal and commercial use) / Premium.
*   **Usage**: Staggeringly large collection of icons spread across numerous styles.

## Best Practices
*   Prioritize JavaScript bundle size efficiency by using inline SVGs packaged as functional React components.
*   Avoid loading monolithic web font files containing thousands of glyphs.
